一个简单的ASN.1和X.509解析器。
ASN.1解析：根据文件路径创建一个Asn1Type对象，然后调用其getStructTree()方法即可获取其结构。
X.509解析：将Asn1Type对象传入Certificate对象中，然后调用Certificate对象的getParseResult()方法即可获取解析结果。
具体用法可以参考Demo类的main()方法。

=========================================================================================================

注意，整个项目都是围绕X.509证书展开的，而X.509仅仅是ASN.1规则的其中一个应用，也就是说在对ASN.1数据的处理中，我仅仅
处理了可能会在X.509证书中出现的数据类型，因此如果使用本项目的ASN.1解析功能解析其它的ASN.1数据，可能会出现异常。
如果需要处理新的数据类型，只需要实现TypeSimpleHandler接口，并将其添加到Asn1Type中的map中即可，基本无需修改代码。
